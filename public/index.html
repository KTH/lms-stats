Fetching data from KTH Canvas LMS

This page is based on realtime data from the Canvas API.
It can take up to about 30 seconds for all graphs to be rendered.

KTH Canvas LMS

All graphs display realtime data unless otherwise stated.

1. KTH Canvas LMS: Daily Statistics

Daily statistics for the KTH Canvas instance regarding pageviews and user participations (interactions with the system).



Views: pageviews within Canvas in any of the categories such as Pages, Modules, Quizzes and Files.
Participations: user interactions with Canvas such as posting a comment, submitting a quiz, submitting assignment.

Instructions
1. Hover pointer for current data for any particular date.
2. Zoom in/out by placing the pointer and use mousewheel/touchpad to zoom in on that area. You can also scroll left/right when you have zoomed in.
3. Toggle which data to highlight/show by hovering/clicking on the legend.

The vertical lines indicate the start of each study period at KTH.

2. KTH Canvas LMS: Ladok course publishing over time

All courses imported from Ladok with at least one person enrolled (teacher and/or student) and their publishing state in Canvas. The data grouped for each study period (P1-4) during the academic year. The bars are stacked with data from each of the five schools at KTH.

Please note that data for this graph is only imported weekly since it is a rather timeconsuming transaction.



Each set of stacked bars:
Courses shows all courses currently imported with at least one user enrolled. For future courses the bar is likely to grow as more courses are added in KOPPS.
Published refer to courses current published for the particular study period regardless of access restriction (closed, institution, public).
Open refer to all published Ladok courses that are published either as open to all users at KTH or Public and thus not requiring a KTH-account to access them.

3. KTH Canvas LMS: Usage

Aggregated statistics for the KTH Canvas instance gathered from 2016-05-10 when the system was first deployed.



4. KTH Canvas LMS: Pageviews per content category

Total number of pageviews grouped by category for the KTH Canvas instance gathered from 2016-05-10 when the system was first deployed.



5. KTH Canvas LMS: Currently published courses/school

Total number courses with the status "published". Please note that not all of these are active since they normally remain published also when the course offering ends. This graph includes all courses (imported courses, manually created courses and sandboxes/s).



6. KTH Canvas LMS: Ladok courses publish status

All imported courses from Ladok and their current publishing state in Canvas. The first view includes also courses without any enrolled users.

Please note that data for this graph is only imported weekly since it is a rather timeconsuming transaction.


Note that a few courses are set as open to both KTH and Public, and hence the total of courses being open to all KTH students can be slightly lower than the sum of the columns to the right..


7. KTH Canvas LMS: Currently published Ladok-courses/school

Total number of courses that are created based on data from Ladok (via KOPPS) with the status "published".



8.KTH Canvas LMS: active teachers/school

Number of users with a teacher role (Teacher, Course responsible, Examinator or TA). Please note that one user can be listed as "teacher" on more than one of the school due to roles being set for each course.



9. KTH Canvas LMS: active students/school

Number of users with a student role (Admitted and/or Registrered) that has logged into the system. Please note that one user can be listed as "student" on more than one of the school due to roles being set on each course.



10. KTH Canvas LMS: uploaded files/school

Total number of files uploaded to KTH Canvas instance.



11. KTH Canvas LMS: student assignement submissions/school

Total number of student submissions in assignments in courses.



12. KTH Canvas LMS: discussion posts/school

Total number of discussion posts in discussion forums in courses.



13. KTH Canvas LMS: Video/Audio objects/school

Total number of video/audio objects created using the native Canvas media recording tool.
